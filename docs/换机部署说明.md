# CourtSim 换机部署说明

从本仓库压缩包解压到新电脑后，按下列步骤即可继续开发或运行。

## 压缩包内有什么

- 完整源码与 **`docs/`** 说明文档：
  - 排障与换机：[BUG修复与排障手记](./BUG修复与排障手记.md)、本说明；
  - 产品文档全集：**[docs/product/](./product/README-文档索引.md)**（规划、需求、交互知识库、美术等）。
- 通常**不包含** `node_modules`、`.next`（体积大，需在新环境重装）。

## 在新电脑上的步骤

1. 安装 **Node.js**（建议 LTS，如 20.x）。
2. 在项目根目录执行：
   ```bash
   npm install
   npx prisma generate
   ```
3. **数据库**：若压缩包里带有 `prisma/dev.db`（或你使用的 SQLite 文件），解压后保持路径与 `.env` 里 `DATABASE_URL` 一致。若没有数据库文件，可执行：
   ```bash
   npx prisma migrate deploy
   ```
4. **环境变量**：将原电脑项目根目录的 **`.env`** 复制过来（勿提交到公开仓库）。至少需配置 `DATABASE_URL` 及模型端点等密钥。
5. 启动开发服务：
   ```bash
   npm run dev
   ```
6. Windows 上若 `npx prisma generate` 报 **EPERM**：先关闭正在运行的 `next dev`、杀毒/同步软件对 `node_modules` 的占用，再重试。

## 排障

详见 [BUG修复与排障手记](./BUG修复与排障手记.md)（Prisma Client 不同步、空 JSON、推演页同步等）。
